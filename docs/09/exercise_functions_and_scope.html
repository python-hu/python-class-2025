演習1：関数の基礎#

演習1-1#

以下の関数 add_numbers は不完全な関数である（そのまま実行してもエラーが出る）．

def add_numbers():
    print(a + b)

2つの引数 a, b を設定し，それらの和を出力するように修正せよ．

また，実際に a に 8，b に 12 を設定して関数を呼び出し，計算結果が正しく表示されることを確認せよ（20 が表示されるはず）．

演習2：関数の実践#

以下の例題を参考に，課題2-1〜2-5の各課題の条件に合う関数を作成せよ． プログラムには，その関数を呼び出す命令文を含めること． 関数に戻り値がある場合は，受け取った戻り値を出力すること（return 文を使って戻り値を設定するかどうかは課題ごとに判断せよ）．

課題を解くためには，前の回で学んだ条件分岐やループ処理に関する知識が必要になるので，不安な人は復習しておこう．

例題#

関数名：　　　get_rectangle_area
引数：　　　　width, height
処理の内容：　引数で渡された幅（width）と高さ（height）の値を持つ長方形の面積を返す．

解答例

def get_rectangle_area(width, height):
    return width * height

result = get_rectangle_area(8, 3) # 関数を呼び出す命令文を書く
print(result) # 戻り値がある場合は，受け取った戻り値を出力する

24

演習3：発展課題#

これ以降は発展的な演習課題となります． 解くことは必須ではありませんが，関数を応用する良い練習になるので，ぜひチャレンジしてみてください．

演習2と同様に，以下の条件に合う関数を作成せよ． 課題を解く上での要件は演習2を参照．

演習3-1#

関数名： get_odd_numbers
引数： numbers（整数が格納されたリスト）
処理の内容： 引数で渡された整数のリスト（numbers）の中に，奇数が何個含まれているかカウントし，そのカウント数を返す．
例： get_odd_numbers([1, 2, 3, 4, 5]) とすると，値 3 を返すようにする．

# 解答用セル

演習3-2#

関数名： get_divisors
引数： number
処理の内容： 引数で渡された正の整数 number の約数をすべて求め，リストとして返す．
例： get_divisors(12) とすると，リスト [1, 2, 3, 4, 6, 12] を返す．
ヒント： 1から number までの整数それぞれで number を割り，余りが0になるものが約数である．

# 解答用セル

演習3-3#

関数名： greet_with_time
引数： name, time="morning"
処理の内容： 引数 name に指定された名前と，time に指定された時間帯に応じた挨拶メッセージを表示する．time には "morning", "afternoon", "evening" のいずれかを指定できる．time はデフォルト引数として定義し，呼び出し時に値が指定されない場合は "morning" が設定されるようにせよ．

実行・出力例1

greet_with_time("Alice", "afternoon")

Good afternoon, Alice!

実行・出力例2（time に値を指定しない場合，time="morning" として扱われる）

greet_with_time("Bob")

Good morning, Bob!

実行・出力例3

greet_with_time("Charlie", "evening")

Good evening, Charlie!

# 解答用セル

演習3-4#

関数名： calculate_product
引数： *numbers
処理の内容： 可変長引数で渡された複数の数値の積を返す．
例： calculate_product(2, 3, 4, 5) とすると，値 120 が返される．
ヒント： 積を求める場合は，初期値を1にして，各数値を順番に掛け算していく．

# 解答用セル

演習3-5#

関数名： apply_operation
引数： numbers（数値が格納されたリスト）, operation（関数）
処理の内容： 第1引数として与えられたリストの各要素に対して，第2引数として与えられた関数を適用し，結果をリストとして返す．
第2引数に指定する関数としては，以下の square 関数を使用せよ．

# 受け取った数値の2乗を返す
def square(value):
    return value ** 2

実行・出力例

numbers = [1, 2, 3, 4, 5]

result = apply_operation(numbers, square)
print(result)

[1, 4, 9, 16, 25]
